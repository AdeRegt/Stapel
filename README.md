# Stapel
## Explaination
Stack based language (compiler tool and virtual machine)

## Required tools
For all:
```Make```
For Windows and Unix:
```gcc```
For WebAssembly:
```emscripten```

## Virtual machine
### Unix
Calls
```./bin/unix/stapel [FILE TO INTERPETATE]```

## Compiler
### Unix
Calls
```./bin/unix/stapelc --input [INPUTFILE] --output [OUTPUTFILE]```
